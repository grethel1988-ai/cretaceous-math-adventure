# 白堊紀冒險營 - 國小數學第一階段線上測驗

這是一個專為國小第一學習階段（一至二年級）學生設計的互動式數學選擇題庫測驗網站。結合了刺激的「白堊紀恐龍冒險」世界觀，讓孩子們在與恐龍小夥伴逃生闖關的過程中，不知不覺完成數學學習指標的自我評量。

## 🎮 遊戲特色
- **沉浸式叢林主題**：精緻的綠野與金黃色調，搭配玻璃擬態面板與逗趣的恐龍夥伴頭像。
- **4 種恐龍夥伴**：三角龍「奇奇」、霸王龍「雷克斯」、甲龍「槌槌」、迅猛龍「小藍」，全程陪伴闖關。
- **50 題動態抽樣關卡**：從 14 個國小數學指標（共 70 題）中 progressive 抽樣 50 題，題庫選項每次載入皆會隨機洗牌。
- **網頁合成音效與特效**：利用 HTML5 Canvas 煙火與 Web Audio API 合成音效，帶來即時反饋且無外部依賴。
- **過關捲軸證書**：自動依成績生成專屬評等徽章，並支援一鍵列印/儲存為 PDF。

## 🚀 如何執行
本專案為純前端靜態網頁（不需安裝後端或資料庫）：
1. 下載或複製本專案至本機。
2. 在瀏覽器中直接點兩下打開 `index.html` 即可開始遊玩！
