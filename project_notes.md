# 專案筆記

## 已完成事項 (已於 2026-07-03 完成)
1. **題庫建立與動態抽樣 (`questions.js`, `app.js`)**
   - 完整解析 14 個能力指標，共 70 道低年級選擇題，設定好正確答案。
   - 實作 progressive sampling，隨機抽出 50 題進行冒險（階段一：17 題；階段二：17 題；階段三：16 題）。
   - 選項 (A, B, C, D) 在每次載入時都會進行隨機打亂，防止學生死記選項位置。
2. **沉浸式叢林視覺設計 (`style.css`, `index.html`)**
   - 設計溫暖綠色與琥珀金的背景，羊皮紙與毛玻璃質感。
   - 加入四隻恐龍夥伴選擇：奇奇（三角龍）、雷克斯（霸王龍）、槌槌（甲龍）、小藍（迅猛龍）。
   - HUD 即時顯示姓名、夥伴、關卡進度、分數，生命值用 10 顆恐龍蛋 `🥚`（扣血變成荷包蛋 `🍳`）顯示。
3. **原生 Web 音效與 Canvas 煙火動畫 (`app.js`)**
   - 使用 Web Audio API 即時合成答對（大調琶音）、答錯（鋸齒波警告聲）、過關勝利（和弦樂章與歡呼聲）音效，免載入外部 mp3。
   - 答對時背景會有七彩紙花（Canvas particles），過關時會有重力粒子煙火動畫。
4. **冒險證書系統 (`index.html`, `app.js`)**
   - 過關後顯示古老捲軸風格的證書，根據分數給予不同勳章（滿分大神、卓越超群等）。
   - 整合 `@media print` 列印與儲存 PDF 樣式，隱藏不必要的按鈕。
