1. **題庫建立與動態抽樣 (`questions.js`, `app.js`)**

   - 設計溫暖綠色與琥珀金的背景，羊皮紙與毛玻璃質感。

   - 答對時背景會有七彩紙花（Canvas particles），過關時會有重力粒子煙火動畫。
4. **冒險證書系統 (`index.html`, `app.js`)**
   - 過關後顯示古老捲軸風格的證書，根據分數給予不同勳章（滿分大神、卓越超群等）。
   - 整合 `@media print` 列印與儲存 PDF 樣式，隱藏不必要的按鈕。
5. **版面尺寸優化 (2026-07-10)**

   - 完成修改並測試後，透過 Git 推送 (push) 更新至 GitHub 遠端儲存庫。
6. **數學分數顯示優化 (2026-07-10)**
   - 在 `app.js` 中實作 `formatFractions` 函式，利用正則表達式自動偵測題目及選項中的斜線分數（如 1/8、7/8、1/10）。
   - 將這些文字分數轉換為直立分式 HTML 結構（包含 `.fraction`, `.num`, `.den`）。
   - 於 `style.css` 中加入對應樣式，利用 `flex` 佈局實現「分子在上、橫線在中間、分母在下」的直立分式效果，以便國小低年級學員理解。
7. **50 題測驗 DOCX 檔案匯出 (2026-07-10)**
   - 撰寫 Python 腳本載入 `questions.js` 題庫，依照遊戲關卡抽取邏輯（第一階段 17 題、第二階段 17 題、第三階段 16 題，共 50 題）進行隨機抽樣。
   - 使用 `python-docx` 生成排版精美的 Word 檔案，依能力指標與關卡分段，每個題目前方皆自動附上對應的數學能力指標說明。
   - **數學分數直立化**：使用 Word 的 Office Math XML (m:oMath) 格式，將題目與選項中所有斜線分數（如 1/8、7/8）在 Word 檔案中皆完美渲染成標準的「直立分式」（分子在上、橫線在中間、分母在下）。
   - **題庫圖片嵌入**：偵測並讀取題庫中的圖像檔案（如天平、日曆、時鐘），自動縮放並以居左縮排方式嵌入於對應題目下方。
   - 在文件結尾新增「參考答案」表格，條列題號、對應指標與正確解答。
   - 已將產生的 `白堊紀冒險營_數學測驗50題.docx` 檔案提交並同步推送到遠端儲存庫。
